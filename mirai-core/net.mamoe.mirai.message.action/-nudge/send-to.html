sendTo

@MiraiExperimentalAPI suspend fun sendTo(receiver: Contact): Boolean

发送戳一戳该成员的消息.

需要 使用协议 MiraiProtocol.ANDROID_PHONE.

Parameters

receiver - 这条 "戳一戳" 消息的接收对象. (不是 "戳" 动作的对象, 而是接收 "A 戳了 B" 这条消息的对象)

Exceptions

UnsupportedOperationException - 当未使用 安卓协议 时抛出

See Also

MemberNudgedEvent

BotNudgedEvent

Return
成功发送时为 true. 若对方禁用 "戳一戳" 功能, 返回 false.
