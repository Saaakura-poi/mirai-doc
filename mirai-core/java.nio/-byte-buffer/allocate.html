allocate

@JvmStatic fun allocate(capacity: Int): ByteBuffer

Allocates a new byte buffer.

The new buffer's position will be zero, its limit will be its capacity, its mark will be undefined, each of its elements will be initialized to zero, and its byte order will be

BIG_ENDIAN.

It will have a backing array, and its array offset will be zero.

Parameters

capacity - The new buffer's capacity, in bytes

Exceptions

IllegalArgumentException - If the capacity is a negative integer

Return
The new byte buffer
